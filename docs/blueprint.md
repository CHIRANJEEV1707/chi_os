# **App Name**: CHIRU-OS Portfolio

## Core Features:

- Boot Sequence Display: Displays a multi-phase boot sequence with textual POST screen, progress bar, and an OS logo, creating an immersive retro start-up experience.
- Retro Desktop Interface: Presents a pixel-art desktop environment with an icon grid for launching applications and a subtle animated canvas wallpaper.
- Interactive Window Management: Allows users to open multiple windows, each with draggable, resizable frames and Z-index stacking for layered interaction.
- Taskbar and Start Menu: A persistent bottom taskbar provides a 'START' button to access a retro Start Menu, showing system information and navigation options.
- Global Sound System: Integrates ambient background sounds and specific sound effects for interactions like opening/closing windows and clicking icons.
- Pixel-Perfect Visual Design: Implements a consistent pixel grid system, custom pixel-art cursors, scanline overlay, CRT vignette, and phosphor flicker effects across the entire interface.

## Style Guidelines:

- The primary color is a vibrant terminal green (#00FF41), symbolizing digital activity and retro computing.
- The background color is a deep, near-black with a subtle green tint (#0A0F0A), providing a strong contrast to the primary green and emphasizing the dark terminal theme.
- An accent color, a glowing chartreuse green (#39FF14), analogous to the primary, is used for highlights and interactive elements, adding visual emphasis and a 'glow' effect.
- For headlines and system labels, use the pixelated sans-serif font 'Press Start 2P'. For longer text content and terminal output, use the monospaced sans-serif 'VT323'. Note: currently only Google Fonts are supported.
- All desktop icons and cursors are custom pixel-art assets, adhering to a 48x48 pixel base for icons and a 16x16 pixel base for cursors, maintaining a consistent retro aesthetic.
- The desktop employs a macOS-style icon grid aligned to the left, and application windows feature fixed titlebars with draggable and resizable bodies, designed to mimic a classic GUI operating system.
- The interface features detailed animations for boot sequences, smooth scaling and pixel dissolve effects for window interactions, and subtle global CRT display simulations such as scanlines and flicker.
